# Art 17 Package

This folder contains documentation, scripts, code and ONNX artifacts related to the ART v7 "art_17" trained model and the pipeline for layers 0–5.

Files included:

- docs/: training and architecture docs (TRAIN_LAYERS.md, etc.)
- scripts/: training and smoke-test scripts (bucle_entrenamiento.sh, train_capa3_5_with_onnx.py)
- code/: TypeScript/Python source files for layers 0–5 and generators
- art_17.onnx, art_17_fixed.onnx, art_17_capa3_5.onnx: ONNX artifacts
- METADATA.sha256: checksums for files in this folder
